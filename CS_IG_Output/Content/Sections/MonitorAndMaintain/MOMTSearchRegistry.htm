Search Registry

You can search your content registry by using Registry Search. This feature uses the same search mechanism as the Gateway, so by using it you can check and verify that particular content will be identified correctly when scanned by the Gateway.

You can search for files or text fragments using the following modes:

  • Normal (Search all content in your registry)

  • Ignore whitelist items

  • Show whitlelist items only

The search will return summary and detailed results. Items returned in the search will be listed in match order, with exact matches at the top. Partial matches will be ordered by number of matches, highest first.

Note:  Due to the nature of the fingerprinting mechanism used by your IG Server, fragments of text with fewer than 150 characters might return unpredictable results.

The list of partial matches might be truncated if a high number of matches are returned by the search.

How do I...

  • Search the registry?
    1. From the Home page, click Tools > Registry Search.
    2. Select the type of item you want to search for and the search mode from the drop down menus.

    3. Depending on your selection:

      1. For a file search, click Choose File, and select the file from the Explore window.
      2. For a text search, click in the text window and enter your search string.

    4. Click Search.

      A list of search results are displayed.
