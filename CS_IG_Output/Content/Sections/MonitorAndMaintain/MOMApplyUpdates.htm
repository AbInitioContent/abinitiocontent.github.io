Install IG Server updates

Updates to your IG Server and supporting platform are provided by Clearswift, and are highlighted on the Updates page when they become available. Updates cover the following:

  • Clearswift IG Server updates
  • Ubuntu package updates

Tip:  We recommend that you do not update your IG Server Ubuntu packages by any other means unless instructed to do so by Clearswift Support.

How do I...

  • Find out what updates have been installed if any new updates are available?

    • From the toolbar, click System and select the Updates tab.

      If new updates are available, an information box will be displayed.

  • Install updates?

    Installing updates is a two step process:

    1. Download updates. Show me
      If you have direct command-line access to your IG Server, you can follow the steps as described below to complete this task. If you only have remote access, you should enable an SSH session first. Show me how

      1. From your IG Server Home page, click System > Security > SSH Access.

        SSH configuration options are displayed.

      2. Click Edit.
      3. Select the Enable SSH access check box, and type the IP or hostname of the SSH client you want to use to connect to your IG Server.
      4. Click Save.

      5. From the SSH client, connect to your IG server.

      Tip:  Leaving SSH access enabled for extended periods might represent a security risk. You should consider disabling SSH when not using it.

      1. From the Clearswift Server Console, select Open Terminal Session.

        A terminal session window is displayed.

      2. Select Configure System > View and Apply Software Updates > Download new updates and press Enter.

      3. Click OK when the download process is complete.

    2. Install updates. Show me
      1. From the Clearswift Server Console, select Configure System > View and Apply Software Updates > Apply updates and press Enter.
      2. Click OK when the installation process is complete.
