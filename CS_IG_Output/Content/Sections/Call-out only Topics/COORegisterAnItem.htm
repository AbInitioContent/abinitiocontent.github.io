Register an item

Use the Register an Item dialog to register and whitelist items.

Registered items will be inspected for matches as part of your organizations policy. Whitelist items and text are not used for matching purposes and are a means of dealing with false negatives, and would typically include document templates and boilerplate text.

Whitelist items are specific to a particular collection and therefore only apply to items that are part of that collection. If you want to whitelist items across all collections, and have sufficient privileges, you can use Common Whitelisting. If you want to use this, note the Common Whitelisting best practice.

You can add single files, or multiple files as part of a zip from the Register an Item dialog. Items need to be added to a Collection. You can add a new collection as part of the registration process but unless you have a Compliance Officer role, you are only allowed to add to collections that you own.

If the Classification feature is enabled, you must supply a classification for registered items. If enabled, a drop down list of available classifications will be displayed.

You can optionally add Custom Properties to registered items. See more on Custom Properties
Custom Properties

These are name and value pairs supplied at registration time that provide extra contextual information for each registered item. In addition, your organization's policy might make use of these properties in order to define different actions depending on the properties of the matching items.

Custom properties have no special meaning to the IG Server; it simply stores them for each item and returns them in query results for matching items. For example, the properties might indicate a particular functional department, for example, Finance. You enter name and value pairs as free format text.

Tell me about...

  • The maximum size of file or item I can register

    Your IG Server allows very large files to be registered and in most cases you should not encounter problems. Files with word counts up to 2.5 million with an average word length of five characters can be supported. This equates to a file size of 15 million bytes.

How do I...

  • Register an item?

    1. From the Home page, click Registry.

    2. Click on the Item Manager tool bar.

      The Register an item dialog is displayed.

    3. Select Registered item from the Item type drop down menu.

    4. Click Choose File.

      The File Explorer window is displayed.

    5. Select the file you want to register.
    6. Select the collection you want to add the file to from the Collection drop down menu. You can create a new collection to add the file to from this dialog.
    7. Optional. If your IG Server has been configured to force item classifications, select the item classification from the Classification drop down menu.
    8. Optional. Add item properties.
    9. Click Register.
  • Whitelist content?

    1. Click Registry.

    2. Click on the Item Manager tool bar.

      The Register an item dialog is displayed.

    3. Select Whitelisted item from the Item type drop down menu.

      Optional, for users with sufficient privileges, select Common whitelisted item.

    4. Depending on what you want to whitelist, select either the Whitelist a file or Whitelist text button.

    5. Depending on your selection in the previous step:

      Whitelist a file:

      Click Choose File and select the file you want to whitelist.

      Whitelist text:

      Enter text to be whitelisted the text box.

    6. Select the collection you want to add the file to from the Collection drop down menu. You can create a new collection to add the file to from this dialog.
    7. Click Register.
