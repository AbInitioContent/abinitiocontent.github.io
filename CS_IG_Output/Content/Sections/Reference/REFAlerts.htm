Alerts

This topic lists alerts that can be raised by your IG Server, and describes steps you can take to deal with them. The list is arranged alphabetically.

At least one system service has stopped unexpectedly

What should I do?

You should attempt to restart the service(s). To do this:

  1. Click System > Services.

    A list of services is displayed.

  2. Click Start for each stopped service.

If you are unable to restart the service and the alert persists contact Clearswift Support.

Diagnostic Logging is currently enabled

What should I do?

Diagnostic logging requires increased system resource and we recommend that it is disabled to prevent poor system performance. To disable diagnostic logging:

  1. Click System > Maintenance.

  2. Click Diagnostics.

    The Diagnostic options are displayed.

  3. Click Edit.
  4. Clear Enable diagnostic logging.
  5. Click Save.

The firewall is not running or is disabled

What should I do?

You should enable the firewall from the console manager.

The IG Server license has not been validated for x days

What should I do?

This alert is raised when your IG Server has connectivity problems with Clearswift. Ensure that your IG Server has a working connection to the Internet and that there are no firewall restrictions.

If you are unable to resolve the problem and the alert persists, contact Clearswift Support.

Licensing has expired

What should I do?

You need to get a new license. Contact your Clearswift representative.

The IG Server server license grace period is due to expire in x days. On expiry, system features will be restricted or unavailable

What should I do?

You need to get a new license. Contact your Clearswift representative.

The IG Server server license is due to expire in x days

What should I do?

You need to get a new license. Contact your Clearswift representative.

The IG Server is unable to contact the following Gateway servers: x

What should I do?

You should check the IG Server configuration settings on the Gateway. To do this:

  1. Log on to the Gateway that the IG Server is unable to contact.

  2. Click Policy > Information Governance.

    The Information Governance page is displayed.

  3. In the IG Server panel, ensure that the Enable IG Server check box is selected and the server details are correct.

The system cpu usage has been consistently high for a sustained period of time (x minutes)

What should I do?

If the alert persists during quiet system times, contact Clearswift Support.

The system disk activity has been consistently high for a sustained period of time (x minutes)

What should I do?

If the alert persists during quiet system times, contact Clearswift Support.

The system disk x is almost full (x used)

What should I do?

Automated processes are in place on your IG Server to manage disk space.

Check that diagnostic logging is disabled.

If the alert persists, contact Clearswift Support.

The system memory usage has been consistently high for a sustained period of time (x minutes)

What should I do?

If the alert persists during quiet system times, contact Clearswift Support.
