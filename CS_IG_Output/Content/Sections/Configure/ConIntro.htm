Configure system options

You configure your IG Server by using the options under the System toolbar option. The following options are available:

System option Purpose
Services

Check status of IG Server services and restart stopped services.
Updates

Check for and install available updates.

View update history.
Policy Configure item classifications, data retention periods, and alert notifications.
License Provide license details and view installed license details.
Network

Configure directory server settings for log on and user management.

Configure mail server settings to be used by your IG Server for email notifications.
Maintenance

Enable diagnostic and remote logging for IG Server troubleshooting.

Configure time for IG Server maintenance.
Security

Add Gateway servers that will communicate with your IG Server.

Add hosts that can access your IG Server using SSH.
