Configure email alert template

Your IG Server is supplied with an email template that is used when administrators are notified of IG Server alerts. You can change any of the text in the template while also making use of tokens. Show me the tokens
Token Description
%ALERTTYPE% Alert type
%ALERTSTATE% Raised or lowered
%ALERTDESCRIPTION% Alert description
%IGSERVERHOMEPAGE% Link to the IG Server Admin UI

To make changes to the template:

If you have direct command-line access to your IG Server, you can follow the steps as described below to complete this task. If you only have remote access, you should enable an SSH session first. Show me how

  1. From your IG Server Home page, click System > Security > SSH Access.

    SSH configuration options are displayed.

  2. Click Edit.
  3. Select the Enable SSH access check box, and type the IP or hostname of the SSH client you want to use to connect to your IG Server.
  4. Click Save.

  5. From the SSH client, connect to your IG server.

Tip:  Leaving SSH access enabled for extended periods might represent a security risk. You should consider disabling SSH when not using it.

  1. From the Clearswift Server Console, select Open Terminal Session.

    A terminal session window is displayed.

  2. Copy the template file for backup purposes. From the command line type the following:

    cp /etc/igp/system/settings/alerts/alert_template.txt /etc/igp/system/settings/alerts/<backup filename>

    where <backup filename> is the name for the backup file.

  3. Use a text editor to open /etc/igp/system/settings/alerts/alert_template.txt.
  4. Make your changes and save the file.
