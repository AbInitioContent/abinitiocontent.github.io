User Interface

The IG Server user interface comprises a number of Centres. Each centre is used for performing different tasks.

Access to the particular centres depends on the role of the user.

Centre name Open to Description
Registry Compliance Officers, IG Administrators, Document Owners

Registering items and reviewing items registered previously.

A Document Owner or IG Administrator can only see their own items. A Compliance Officer can see all items.
Users Compliance Officers, IG Administrators

Managing which users can access the system, and defining their roles.

Viewing and deleting active sessions
System Compliance Officers, IG Administrators, Service Providers For configuring system settings, installing software updates, viewing system logs, monitoring system health, license validation, and controlling services.
Reports Compliance Officers, Line Managers

View predefined reports including item matches, user activity, registration activity.

For reporting on the movement and whereabouts of data (Compliance Officer only).
Tools Compliance Officers, Line Managers

Perform registry search to verify that registered items will be identified by the Gateway.
