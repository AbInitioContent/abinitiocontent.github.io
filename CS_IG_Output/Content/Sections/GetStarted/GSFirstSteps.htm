First Steps

This topic provides background information and guidance on getting started with your IG Server.

  1. Overview

    Get an overview of the IG platform, including component descriptions and architectural diagrams. Go

  2. Familiarise yourself with key IG Server concepts

    Understanding some of the key concepts will help you work more efficiently with your IG server. Go

  3. Install updates

    When you first install your IG Server you should check for and install any updates. By installing updates, you will ensure that your IG server has up-to-date security protection. Go

  4. Check (and update) system settings

    Configuration settings are applied at the time of installation, but you can update these settings at any time. Go

  5. Add users and groups

    You can create users and groups depending on the demands of your policy. User and group creation is a task for IG Administrator or Compliance Officer roles. Go

  6. Protect your content

    Register content you want to protect by creating collections and adding items to these collections. All IG roles can create collections and add items. Go

  7. Configure your Gateway

    To make sure that movements of registered documents are tracked and reported as part of your organizations policy, you need to configure the Gateway you are using with your IG Server. This includes configuring connection settings and applying IG content rules. Details of these tasks are in the online help provided with your Gateway.
