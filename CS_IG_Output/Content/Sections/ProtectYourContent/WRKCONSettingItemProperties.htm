Set item properties

You can set item properties both before and after item registration. You can not add item properties to whitelisted items.

  Some item properties can not be changed. For example, the user who first registered the item, and time of registration.

Custom Properties

These are name and value pairs supplied at registration time that provide extra contextual information for each registered item. In addition, your organization's policy might make use of these properties in order to define different actions depending on the properties of the matching items.

Custom properties have no special meaning to the IG Server; it simply stores them for each item and returns them in query results for matching items. For example, the properties might indicate a particular functional department, for example, Finance. You enter name and value pairs as free format text.

How do I...

  • Set item properties when registering a new item?

    1. Click Registry.

    2. Click + on the Item Manager tool bar.

      The Register an item dialog is displayed.

    3. Select Registered item from the Item type drop down menu.

    4. Click Choose File.

      The File Explorer window is displayed.

    5. Select the file you want to register.
    6. Select the collection you want to add the file to from the Collection drop down menu. You can create a new collection to add the file to from this dialog.
    7. Optional. If your IG Server has been configured to force item classifications, select the item classification from the Classification drop down menu.

    8. Click Add properties.

      The Add properties dialog is expanded.

    9. Click + to add name and value pairs. Click Add.
    10. Click Register.
  • Set item properties for a registered item?

    1. In the Registry centre, select the file you want to set properties for and click the EDIT icon.

      Item details are displayed.

    2. Click the Edit button and make your changes.
    3. Click Save.
