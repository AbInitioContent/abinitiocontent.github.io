Register content

When you register content on your IG Server, you need to add it to a Collection. You can add content to an existing collection, or create a new one during the process of adding the content. You can register files individually or as part of an archive.

Note:  You can only add content to collections that you have access to. If you have selected a collection that you do not have access to, you will be prompted to create a new collection when you register the content.

Tell me about...

  • What happens when I register an archive file

    Each file, including the archive itself, will be registered separately within the collection.

  • The maximum size of file or item I can register

    Your IG Server allows very large files to be registered and in most cases you should not encounter problems. Files with word counts up to 2.5 million with an average word length of five characters can be supported. This equates to a file size of 15 million bytes.

How do I...

  • Add a new collection?

    You can add a new collection in two ways:

    • By adding the collection from the Collection panel. You can do this without having to register content at the same time. Show me
      1. Click Registry.
      2. Select the user or group that you want to add the collection to from the drop down menu.

      3. Select Create a collection from the Collections drop down menu.

        The Create a collection dialog is displayed.

      4. Enter the collection name and click Create.
    • By adding the collection at the same time as registering content. Show me

      1. Click on the Item Manager bar.

        The Register an item dialog is displayed.

      2. After you have selected the file to upload, select Create a collection from the Collection drop down menu.
      3. Enter the collection name and click Create.
  • Register content?

    1. From the Home page, click Registry.

    2. Click on the Item Manager tool bar.

      The Register an item dialog is displayed.

    3. Select Registered item from the Item type drop down menu.

    4. Click Choose File.

      The File Explorer window is displayed.

    5. Select the file you want to register.
    6. Select the collection you want to add the file to from the Collection drop down menu. You can create a new collection to add the file to from this dialog.
    7. Optional. If your IG Server has been configured to force item classifications, select the item classification from the Classification drop down menu.
    8. Optional. Add item properties.
    9. Click Register.
  • Whitelist content?

    1. Click Registry.

    2. Click on the Item Manager tool bar.

      The Register an item dialog is displayed.

    3. Select Whitelisted item from the Item type drop down menu.

      Optional, for users with sufficient privileges, select Common whitelisted item.

    4. Depending on what you want to whitelist, select either the Whitelist a file or Whitelist text button.

    5. Depending on your selection in the previous step:

      Whitelist a file:

      Click Choose File and select the file you want to whitelist.

      Whitelist text:

      Enter text to be whitelisted the text box.

    6. Select the collection you want to add the file to from the Collection drop down menu. You can create a new collection to add the file to from this dialog.
    7. Click Register.

Tip:  You can check to ensure that registered content will be identified by your Gateway security policy by using the Registry Search feature.
