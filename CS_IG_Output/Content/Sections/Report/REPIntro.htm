Run reports

Your IG Server is supplied with a number of pre-configured reports covering the following:

  • Gateway activity

    Includes data tracking, tracking and query trends.

  • Incidents

    Includes data on Gateway query matches by owner and classification.

  • Registry

    Reports on registered items by user and classification.

  • User activity

    Reports on IG Server user activity.

How do I...

  • Run reports?

    From the Home page, click Reports.

    The Reports page is displayed. The list of available reports are grouped in the left panel. Click on the grouping to expand the list and select a report to see a description and run the report.

    Tip:  Use a combination of report parameters and column sorting where available to fine tune your report output.

    Some reports display column or pie chart outputs. Click on chart segments to drill down into the report.
